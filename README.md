# 🌐 Web CEXPO 26

Selamat datang di repositori resmi untuk website CEXPO 2026. Proyek ini berisi kode sumber lengkap (HTML, CSS, JavaScript) untuk halaman web statis pameran tersebut.

Website ini di-deploy secara otomatis menggunakan GitHub Pages dan dapat diakses melalui URL berikut:
[**[faiz-jihad.github.io](faiz-jihad.github.io)**](https://cexpo2026.netlify.app/)

## 📝 Deskripsi Proyek

Repositori ini berfungsi sebagai *landing page* dan portal informasi untuk CEXPO 2026. Tujuannya adalah menyediakan detail acara, jadwal, informasi pembicara, dan panduan pendaftaran bagi peserta dan pengunjung.

## 📁 Struktur Folder

Struktur file utama dalam proyek ini adalah:


├── index.html # Halaman utama website
├── css/ # Folder untuk file CSS (stylesheet)
├── js/ # Folder untuk file JavaScript (interaktivitas)
├── img/ # Folder untuk aset gambar dan media
├── LICENSE # Informasi lisensi (jika ada)
└── README.md # File yang sedang Anda baca ini

## 🚀 Cara Menjalankan Secara Lokal

Jika Anda ingin menjalankan atau menguji website ini di komputer lokal Anda:

1.  **Clone repositori ini:**
    ```bash
    git clone github.com
    ```

2.  **Buka folder proyek:**
    ```bash
    cd Webcexpo26
    ```

3.  **Buka file `index.html`** menggunakan browser web favorit Anda (seperti Google Chrome, Firefox, Safari).
